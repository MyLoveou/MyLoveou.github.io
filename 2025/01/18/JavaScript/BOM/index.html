BOM

BOM是浏览器对象模型（Browser Object Model）的缩写，它提供了独立于内容而与浏览器窗口进行交互的对象。BOM的核心对象是window，它表示浏览器的一个实例。BOM由一系列相关的对象构成，并且每个对象都提供了很多方法与属性。以下是BOM中一些主要对象及其常用方法：

一、window对象

  • alert()：弹出一个警告框，显示指定的文本和OK按钮。例如window.alert("这是一个警告框！")。
  • confirm()：弹出一个带有确定和取消按钮的对话框，并返回用户的选择（true或false）。例如var result = window.confirm("你确定要继续吗？")。
  • prompt()：弹出一个带有文本输入字段和确定及取消按钮的对话框，并返回用户输入的文本（如果用户点击确定）或null（如果用户点击取消）。例如var name = window.prompt("请输入你的名字", "Harry Potter")。
  • open()：用于打开一个新的浏览器窗口。例如window.open("https://www.example.com")。
  • close()：用于关闭当前浏览器窗口。例如window.close()。
  • setTimeout()：用于在指定的毫秒数后执行函数。例如var timerId = window.setTimeout(function() {console.log("5秒后执行此函数")}, 5000)。
  • clearTimeout()：用于取消之前设置的定时器。例如window.clearTimeout(timerId)。
  • setInterval()：用于每隔指定的毫秒数重复执行函数。例如var intervalId = window.setInterval(function() {console.log("每秒执行此函数")}, 1000)。
  • clearInterval()：用于取消之前设置的间隔定时器。例如window.clearInterval(intervalId)。

二、location对象

  • href：用于获取或设置窗口的URL。例如window.location.href = "https://www.example.com"用于设置URL，var currentURL = window.location.href用于获取当前URL。
  • hostname：用于获取URL的主机名。例如var hostname = window.location.hostname。
  • pathname：用于获取URL的路径。例如var pathname = window.location.pathname。
  • search：用于获取URL的查询字符串。例如var search = window.location.search。
  • hash：用于获取URL的锚点。例如var hash = window.location.hash。

三、history对象

  • back()：用于后退一页。例如window.history.back()。
  • forward()：用于前进一页。例如window.history.forward()。
  • go(n)：用于加载历史列表中的特定页面，n为要加载的页面在历史列表中的相对位置。例如window.history.go(-1)用于后退一页，window.history.go(1)用于前进一页。

四、navigator对象

  • appName：用于获取浏览器名称。例如var browserName = window.navigator.appName。
  • appVersion：用于获取浏览器版本。例如var browserVersion = window.navigator.appVersion。

五、screen对象

  • width：用于获取屏幕宽度。例如var screenWidth = window.screen.width。
  • height：用于获取屏幕高度。例如var screenHeight = window.screen.height。
