DOM

什么是DOM

DOM 是“Document Object Model（文档对象模型）”的缩写，它是 HTML 和 XML 文档的编程接口，提供了对文档的结构化表示，并且可以使用诸如 JavaScript 等编程语言来访问和操作文档中的元素、属性和事件等。以下是关于 DOM 的详细介绍：

DOM 的结构

  • 节点（Node）

    • 元素节点（Element Node）：对应于 HTML 或 XML 文档中的元素，如 <div>、<p> 等。元素节点是 DOM 树的主要组成部分，可以包含其他元素节点、文本节点等。
    • 属性节点（Attribute Node）：定义了元素的属性，如 <img src="image.jpg"> 中的 src 就是一个属性节点。不过在 DOM 中，属性通常被视为元素节点的一部分，而不是独立的节点。
    • 文本节点（Text Node）：包含元素的文本内容。例如在 <p>Hello, world!</p> 中，“Hello, world!” 就是一个文本节点。
    • 注释节点（Comment Node）：对应于文档中的注释，如 <!-- 这是一个注释 -->。
    • 文档节点（Document Node）：是 DOM 树的根节点，代表整个文档。在 JavaScript 中，可以通过 document 对象来访问文档节点。

  • DOM 树（DOM Tree）

    • DOM 将文档表示为一个倒立的树形结构，其中每个节点都是树的一个组成部分。文档节点是树的根节点，其他节点如元素节点、文本节点等通过父子、兄弟等关系相互连接。例如对于以下 HTML 结构：

DOM 的操作

  • 访问元素

    • 通过 ID 获取元素：document.getElementById('elementId')，返回具有指定 ID 的元素。
    • 通过类名获取元素：document.getElementsByClassName('className')，返回一个包含所有具有指定类名的元素的 HTMLCollection 对象。
    • 通过标签名获取元素：document.getElementsByTagName('tagName')，返回一个包含所有具有指定标签名的元素的 HTMLCollection 对象。
    • 通过选择器获取元素：document.querySelector('.className') 或 document.querySelector('#elementId') 等，返回文档中匹配指定选择器的第一个元素；document.querySelectorAll('.className') 等，返回一个包含所有匹配指定选择器的元素的 NodeList 对象。

  • 创建元素

    • document.createElement('tagName')：创建一个指定标签名的新元素。
    • document.createTextNode('text')：创建一个包含指定文本的新文本节点。

  • 插入元素

    • element.appendChild(child)：将一个子元素添加到指定元素的末尾。
    • element.insertBefore(newElement, referenceElement)：将新元素插入到指定元素的 referenceElement 子元素之前。

  • 删除元素

    • element.removeChild(child)：从指定元素中删除一个子元素。

  • 修改元素

    • 修改元素的属性：element.setAttribute('attributeName', 'value')。

    • 修改元素属性：element.className = '其他CSS选择器'，覆盖原有属性类名

      • 可以使用classlist操纵CSS
        • element.classlist.add('其他CSS选择器')：追加
        • element.classlist.remove('其他CSS选择器')：删除
        • element.classlist。toggle('其他CSS选择器')：切换，如果有这个属性，就删除，反之，则添加

    • 修改元素的内容：element.innerHTML = '新的内容' 或 element.textContent = '新的内容'。

      • innerHTML 属性用于获取或设置元素的 HTML 内容，它会解析内容中的 HTML 标签。当使用这个属性输出DOM对象的内容是，会将HTML标签也进行输出

      • 与innerHTML属性不同，innerText属性用于获取或设置元素的文本内容，它不会解析内容中的 HTML 标签。

  • 自定义属性

    • 在HTML标签中使用”data-”作为开头，同时使用dataset获取自定义属性，获取到的是一个字典。

  • 事件处理

    • 添加事件监听器：element.addEventListener(eventclass, function,yesOrno)，为元素添加一个事件监听器，当指定事件发生时，执行回调函数。

      • eventclass：事件类型
      • function：要执行的函数
      • yesOrno：是否进行捕获，默认为true

    • 移除事件监听器：element.removeEventListener('eventName', function)，移除元素上之前添加的事件监听器。

    • 事件类型

      • 鼠标事件
        • mouseenter：鼠标是否移入该DOM对象
        • mouseleave：鼠标是否移出该DOM对象
        • click：鼠标单击
        • dbclick：鼠标双击
      • 键盘事件
        • keydown：键盘按下
        • keyup：键盘抬起
      • 表单事件
        • input：文本框输入
      • 焦点事件
        • focus：获得焦点
        • blur：失去焦点

    • 任意事件类型被触发时与事件相关的信息会被以对象的形式记录下来，我们称这个对象为事件对象。事件回调函数的【第1个参数】即所谓的事件对象，通常习惯性的将这个对象命名为 event、ev 、ev 。

    • 这个对象包含的属性

      1. ev.type 当前事件的类型
      2. ev.clientX/Y 光标相对浏览器窗口的位置
      3. ev.offsetX/Y 光标相于当前 DOM 元素的位置

DOM 的应用

  • 页面交互：通过 DOM 操作，可以实现各种页面交互效果，如点击按钮显示隐藏内容、输入框内容验证、拖拽元素等。
  • 动态内容更新：根据用户操作或数据变化，动态地创建、修改或删除页面元素，从而更新页面内容，无需重新加载整个页面。
  • 与 CSS 交互：可以使用 DOM 来获取或修改元素的样式，实现动态的样式效果，如改变元素的颜色、大小、位置等。
  • 数据可视化：结合数据和 DOM 操作，可以生成各种图表、图形等数据可视化内容，为用户提供更直观的信息展示。
