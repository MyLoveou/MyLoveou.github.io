函数

一、什么是函数

在JavaScript 使用关键字 function 定义函数。不同于其他编程语言中的函数定义，在Javascript中函数描述为一个对象更为准确，JavaScript函数有对象和方法，而且函数定义方法多样。

二、函数分类

1、基本函数

1、语法

2、特点

  • 可以在函数被声明前被调用
  • 函数名整个作用域都有效

2、匿名函数

1、语法

2、特点

  • 由于没有函数名，不能在函数外部被调用。

3、箭头函数

1、语法

2、特点

  • 没有自己的 this，this 的值来源于外围最近一层非箭头函数的上下文。
  • 不绑定 arguments 对象，只能使用 rest 参数代替。
  • 不能用作构造函数，即不能使用 new 关键字调用。
  • 简化了函数的写法，如果函数体只有一条语句，可以省略花括号，并且该语句的返回值自动成为函数的返回值。例如 const double = x => x * 2;

4、方法定义

1、语法

2、特点

  • 主要用于对象的方法定义，使代码更加简洁易读。
  • 方法内部的 this 指向该对象。

5、立即执行函数

1、语法

2、特点

  • 文件一保存就执行这个代码
  • 用于创建一个封闭的作用域，避免变量污染全局作用域。

6、回调函数

1、语法

2、示例

3、特点

  • 作为参数传递给另一个函数，并在该函数内部被调用。
  • 常用于处理异步操作，如事件处理、异步请求等。
  • 如果回调函数执行不当，可能会导致回调地狱（Callback Hell），使代码难以维护。

7、间歇函数

1、语法

  • function：要周期性执行的函数。
  • delay：每次执行之间的时间间隔，以毫秒为单位。
  • arg1, arg2, ...：可选参数，这些参数会传递给要执行的函数。
